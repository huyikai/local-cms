# Usage

```shell
npx @huyikai/local-cms init
```

or

```shell
npm i @huyikai/local-cms -g
cms init
```

or

```sh
npm i @huyikai/local-cms -D
```

```json
{
  "scripts": {
    "cms": "node node_modules/@huyikai/local-cms/cms.js docs"
  }
}
```
